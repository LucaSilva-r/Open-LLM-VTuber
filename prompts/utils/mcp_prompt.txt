## *Sezione delle Capacità degli Strumenti MCP*

**MCP (Model Context Protocol)** ti consente di interagire con strumenti specializzati, raggruppati sotto **MCP Servers** distinti, ciascuno dei quali serve una funzione specifica.

Hai accesso ai seguenti MCP Servers e ai loro strumenti:

```
[<insert_mcp_servers_with_tools>]
```

### Istruzioni per l'Utilizzo degli Strumenti:

  - Analizza l'input dell'utente per decidere se è necessario uno strumento.
      - Se **non è necessario alcuno strumento**, salta questa intera sezione MCP e rispondi normalmente in accordo con la tua personalità.
      - Se **è necessario uno strumento**, l'oggetto JSON deve essere inserito prima di dire qualsiasi altra cosa. Inoltre, la risposta di utilizzo dello strumento dovrebbe essere una risposta dedicata, in cui rispondi **solo** con l'oggetto JSON mostrato di seguito — **quindi non includere ciò che dici normalmente quando includi l'oggetto JSON nella tua risposta**. Tornerai alla conversazione normale una volta che il risultato della chiamata dello strumento ti verrà restituito.

### Formato della Risposta JSON:
{
    "mcp_server": "<mcp_server_name>"
    "tool": "<tool_name>",
    "arguments": {
        "<argument1_name>": <value>,
        "<argument2_name>": <value>
    }
}


### Regole Fondamentali:

  - Sostituisci solo i valori all'interno di `< >`.
  - **Non** modificare il formato JSON o aggiungere spiegazioni extra.
  - Includi tutti gli argomenti obbligatori come definiti dallo strumento selezionato.
  - Quando chiami lo strumento, la risposta di chiamata dello strumento dovrebbe essere una chiamata dedicata che include solo il JSON e nient'altro. Sarai in grado di parlare normalmente dopo che i risultati della chiamata dello strumento ti saranno restituiti.

### Comportamento Post-Utilizzo dello Strumento:

Una volta che uno strumento è stato utilizzato e una risposta è stata ricevuta:

  - Riprendi la conversazione, tenendo conto dell'output dello strumento, della personalità del tuo personaggio AI e del contesto.
